# Согласие на обработку данных с использованием сервиса Яндекс Метрика

Настоящим я, пользователь сайта webshopstudio.ru, свободно, своей волей и в своем интересе даю согласие владельцу сайта webshopstudio.ru, далее — Оператор, на обработку данных, собираемых с использованием сервиса веб-аналитики Яндекс Метрика.

## 1. Цель обработки данных

Данные обрабатываются в целях анализа посещаемости сайта, оценки эффективности страниц, улучшения структуры сайта, повышения удобства использования сайта и анализа действий пользователей на сайте.

## 2. Какие данные могут обрабатываться

С использованием сервиса Яндекс Метрика могут обрабатываться следующие данные:

- cookie-файлы;
- IP-адрес;
- сведения о браузере;
- сведения об устройстве пользователя;
- сведения об операционной системе;
- адрес страницы, с которой пользователь перешел на сайт;
- страницы сайта, которые посещает пользователь;
- дата и время посещения сайта;
- действия пользователя на сайте, включая просмотры страниц, клики, прокрутку страниц и взаимодействие с формами;
- иные технические данные, автоматически передаваемые браузером пользователя.

## 3. Используемый сервис

Для обработки данных используется сервис веб-аналитики Яндекс Метрика, предоставляемый ООО «Яндекс».

Яндекс Метрика использует cookie-файлы и иные технические данные для формирования статистики посещаемости сайта и анализа поведения пользователей.

## 4. Действия с данными

В рамках настоящего согласия Оператор вправе осуществлять сбор, запись, систематизацию, накопление, хранение, уточнение, использование, обезличивание, передачу, удаление и уничтожение данных.

## 5. Передача данных

Данные, собираемые с использованием Яндекс Метрики, могут передаваться ООО «Яндекс» для обработки в целях веб-аналитики.

Оператор не использует данные Яндекс Метрики для установления личности конкретного пользователя без необходимости и без отдельного правового основания.

## 6. Срок действия согласия

Настоящее согласие действует с момента его предоставления пользователем и до момента его отзыва.

## 7. Отказ от обработки данных

Пользователь может отказаться от обработки данных с использованием cookie-файлов и Яндекс Метрики, не нажимая кнопку согласия в cookie-баннере, а также ограничив или запретив использование cookie-файлов в настройках своего браузера.

При отключении cookie-файлов отдельные функции сайта могут работать некорректно.

## 8. Отзыв согласия

Пользователь может отозвать настоящее согласие, очистив cookie-файлы сайта в настройках браузера или направив обращение Оператору по электронной почте: ivanbogachenco@gmail.com.

## 9. Подтверждение согласия

Нажимая кнопку согласия в cookie-баннере на сайте webshopstudio.ru, пользователь подтверждает, что ознакомился с настоящим согласием, понимает его содержание и дает согласие на обработку данных с использованием сервиса Яндекс Метрика.
